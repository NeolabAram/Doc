carfile

terminal 에서 carfile을 생성합니다.

$ cd $ touch Cartfile $ open Cartfile -a Xcode

carfile의 형태는 아래와 같습니다.

github realm/realm-cocoa github drmohundro/SWXMLHash ~> 4.0

cartfile 위치에서 carthage update 실행

$ cartfile update
